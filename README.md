# EM::Cron

Simple gem for running code on a schedule using the crontab syntax.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'em-cron'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install em-cron
